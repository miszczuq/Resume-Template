Projekty i doświaczenie

CartRacing

Projekt mający na celu stworzenie aplikacji dla firmy zajmującej się wynajmowaniem gokartów oraz organizowaniem wyścigów na torze. Projekt został podzielony na dwie aplikacje frontendową i backendową. Backend został napisany w Node.js z wykorzystaniem ORM Sequelize. Aplikacja frontendowa została napisana w React.js. W projekcie zostały wykorzystane obrazy dockerowe MongoDB, MySQL i phpMyAdmin

Discordziak

Aplikacja, którą w wolnym czasie współtworzę ze znajomymi. Zadaniem jej jest integracja z API Discord'a. Założeniem aplikacji jest stworzenie serwisu, na którym użytkownicy mogą dla swoich kont zbierać punkty i doświadczenie poprzez różne aktywności na serwerze discorda, które później mogą być wymieniane na nagrody np. role lub akcje na serwerze discorda. Przy tworzeniu aplikacji zajmowałem się częścią Backendową. Backend został napisany w Spring Boot Rejestracja i logowanie zrealizowaliśmy za pomocą OAuth2 Discorda. Schemat bazy danych został stworzony przy pomocy Hibernate'a Backend został stworzony jako REST API
